# 1. Mục tiêu
Cung cấp "hệ gen" phong cách cho AI trước khi dịch, đảm bảo xưng hô đồng nhất và từ vựng phù hợp thể loại.

---

# 2. Các chế độ hoạt động (UI Modes)

## 2.1 View Mode (Summary)
- Hiển thị các Tag (Genre/Tone) đã chọn dưới dạng chip.
- Hiển thị thông tin tiêu đề đã được Việt hóa.
- Thanh trượt Humor Level (0-10) để điều chỉnh độ "thoát ý".

## 2.2 Edit Mode (Manual Configuration)
- Cho phép user chọn thủ công từ danh sách Suggested Genres/Tones (Tu tiên, Đô thị, Hài hước, v.v.).
- Bấm "Done Editing" để lưu trạng thái.

---

# 3. Discovery Flow (AI-Driven)
User có thể bấm **Analyze Title**:
1. AI trích xuất tiêu đề thực tế từ tên file.
2. Dịch tiêu đề sang Tiếng Việt.
3. Dự đoán các thể loại và tông giọng phù hợp.
4. Tự động nạp vào Preset.

---

# 4. Persistence
Hỗ trợ **Import/Export Preset** dưới định dạng `.json` để tái sử dụng cho các tập phim khác trong cùng series.
